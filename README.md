# Artemis II Calendar (iCal / WebCal)

This repository provides a continuously maintainable **iCalendar (ICS) feed** for the
**NASA Artemis II crewed Moon mission**, focused on **officially published launch opportunities**.

The calendar is intended for **subscription**, not one-time import, so updates propagate automatically.
